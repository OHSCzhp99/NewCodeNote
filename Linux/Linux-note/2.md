### 1. Linux的目录结构
```python
① `/`，根目录是最顶级的目录了
② /home/itheima/a.txt，home一般放用户
例如win系统中的 c:\user\zhp\a.txt

蓝色：代表目录
白色：代表文件
绿色：代表可执行文、脚本文件
红色：代表压缩文件或包文件
浅蓝色：代表链接文件、音视频，这些文件是指向其他文件或目录的引用。
黄色：代表设备文件，包括块设备文件和字符设备文件。这些文件代表了系统中的物理设备或虚拟设备。
灰色：代表其他文件类型，这可能包括特殊文件或系统保留的文件类型。
红色闪烁：表示链接的文件有问题。
```

### 2. 常用命令
```python
1.pwd 打印当前目录路径
2.ls  打印当前目录文件、文件夹内容
-a 全部文件，包括隐藏文件
-l 细节，包括日期、权限等
-h 显示大小
------------------------
3.cd  进入目录 
cd /  转到根目录
cd ~  转到用户目录
cd /home/zhp 转到用户zhp目录下 - 绝对路径
------------------------
3.mkdir ./test   创建目录 
4.touch test.txt 创建文件 
5.cat test.txt   查看文件内容
6.more test.txt  查看文件内容 空格翻页、q退出
7.cp test.txt test2.txt 复制test.txt 到当前文件夹名为test2
cp -r itheima itheima2  复制文件夹 加上 -r
8.mv test.txt Desktop/  移动到文件或文件夹到桌面，如果路径不存在则是改名效果
9.rm test.txt           删除文件，文件夹要加上 -r
rm -r itheima1 itheima2  删除多个空格隔开
rm test*                删除多个文件，通配符*，位置可变
rm -f                   强制删除不提示，进入root删除会提示，用-f则不提示
------------------------
1.which cd             查找'命令'的路径
2.find / -name "test"  搜索从根目录下名为test的文件
find / -name "*test"   可使用通配符
find / -size -10K      搜索大小 小于10K的文件
find / -size +100M     搜索大小 大于100M的文件
3.grep "itheima" test.txt  过滤显示出文件中有itheima的行，加上-n 显示第几行数
4.wc test.txt          统计test文件 行数、单词数、字节大小
5.|                    管道符，将左边的结果作为右边的输入
------------------------
1.echo hello Linux       输出内容
echo `pwd`               被``包围会作为命令结果输出
echo "hello" > test.txt  将左边内容覆盖到test文件
echo "hello" >> test.txt 将左边内容覆盖追加到test文件
2.tail -20 test.txt      查看文件倒数20行
tail -f test.txt         持续实时追踪，加上 -f 可打开两个客户端试下
------------------------
1.vim hello.txt     编辑文件，没有就自动创建，进入命令模式
i                   插入
:wq                 :是命令模式，wq是保存并退出
------------------------
1.su - root             进入root权限
exit 或者 ctrl+d        退回
2.sudo                  临时root权限命令，网上查步骤设置
------------------------
前提是进入root
1.groupadd 用户组名     添加用户组
2.groupdel 用户组名     删除用户组
3.useradd 用户名 -g itheima -d /home/itheima 
添加用户，-g指明用户组，没有则默认和用户同名， -d 指明用户HOME路径
4.userdel 用户名 -r           删除用户，-r删除用户的HOME目录，不加则不删除目录
5.id 用户名                   如果不指明用户名则查看自己
6.usermod -aG itheima test2   将test2用户加入到itheima用户组里
7.getent passwd               查看系统有哪些用户

```

### 5. Linux下载命令 和 压缩包
```python
rpm 全称是Red Hat Package Manager（Red Hat包管理器），将所需要的软件安装到Linux主机上
yum可以自动的处理依赖性关系，并且一次安装所有依赖的软件包，无须繁琐地一次次下载、安装

Linux采用 yum、rpm 来进行下载
Linux常用的压缩包后缀：tar、zip、gz、gzip
--------------------------
常用的组合为：
tar压缩：
tar -cvf test.tar 1.txt 2.txt 3.txt      将压缩到test.tar文件内
tar -zcvf test.tar.gz 1.txt 2.txt 3.txt  将压缩到test.tar.gz文件内

tar解压：
tar -xvf test.tar  解压test.tar到当前目录
tar -xvf test.tar -C /home/itheima  解压test.tar到指定目录
tar -zxvf test.tar -C /home/itheima 解压test.tar.gz到指定目录

-c 压缩模式
-x 解压模式
-v 显示压缩、解压过程，用于查看进度
-f 要创建的文件，或要解压的文件,-f选项必须在所有选项中位置处于最后一个
-z gzip模式，不使用-z就是普通的tarball格式
-c 选择解压的目的地，用于解压模式
--------------------------
zip压缩：
zip test.zip a.txt b.txt c.txt  将压缩到test.zip里
zip -r test.zip itheima b.txt c.txt  将压缩到test.zip里，如果包含文件夹则加- r

zip解压：
unzip test.zip 将test.zip解压到当前目录
unzip test.zip -d /home/itheima 将test.zip解压到指定目录
```
