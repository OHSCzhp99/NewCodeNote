## Git
---
### 1. Git上传命令：
```java
在Github或者Gitee创建仓库后：

上传git：
1、在本地创建一个版本库（即文件夹），通过git init，把它变成Git仓库；
2、把项目复制到这个文件夹里面，再通过git add *，把项目添加到暂存区；
3、再通过git commit -m "版本说明"，把项目提交到本地仓库；
4、通过git remote add origin 远程仓库地址，将本地仓库和远程仓库进行关联；
5、最后通过git push -u origin master把本地仓库的项目推送到远程仓库（也就是Github或Gitee）上；

如果在第4步关联了错误的git仓库，可以通过一下两行代码删除，然后重新执行第4步
查看现有远程仓库的信息（git remote -v）；
删除现有的远程仓库信息（git remote rm origin）；


更新git：
上面的省去第1步和第4步
```
