### 5. 对象 - 注解
```python
注解的形式：变量: 数据类型

#给变量注解：
var_1: int = 10
var_2: str = "itheima"
var_3: bool = True

#给类对象注解：
class Student:
    pass

stu: Student = Student()

#基础容器类型注解：
my_list: list = [1,2,3]
my_tuple: tuple = (1,2,3)
my_dict: dict = {"ithiema"}
#容器类型详细注解：
my_list: list[int] = [1,2,3]
y_tuple: tuple[int,str,bool] = (1,"itheima",True)
my_dict: dict[str,int] = {"ithiema": 666}

#用注释注解：
var_1 = random.randint(1, 10) # type: int

#对方法参数类型注解：
def add(x: int, y: int):
    return x+y
def func(data: list) -> list: # ->是对返回值的类型注解
    return data

#联合注解：表示可以是多种数据类型
from typing import Union  #union联合注解要导包

my_list: list[Union[int,str]] = [1,2,""itheima]

def func(data: Union[int,str]) -> Union[int,str]:
    pass
```

### 6. 对象 - 多态
```python
多态是父类做顶层设计，子类做具体实现

#定义一个抽象父类，让子类去写具体的实现方法
class AC:
    def cool_wind(self): #制冷方法
        pass
    def hot_wind(self): #制热方法
        pass
    def swing_l_r(self): #左右摆风
        pass

class Midea_AC(AC):
    def cool_wind(self):
        print("美的空调制冷")
    def hot_wind(self): 
        print("美的空调制热")
    def swing_l_r(self): 
        print("美的空调左右摆风")

class GREE_AC(AC):
    def cool_wind(self):
        print("格力空调制冷")
    def hot_wind(self): 
        print("格力空调制热")
    def swing_l_r(self): 
        print("格力空调左右摆风")

#定义一个制冷方法，将不同子类作为传参，将输出不同子类的各自方法
def make_cool(ac: AC):
    ac.cool_wind()

#将子类对象传给方法
make_cool(Midea_AC())
make_cool(GREE_AC())
```
