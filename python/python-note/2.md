### 4. range语句
```python
1.获得一个数字序列
① range(num) 从0开始，到num（不包括num）
② range(num1,num2) 从0开始，到num2（不包括num2）
③ range(num1,num2,step) 从0开始，到num2（不包括num2），步长以step值为准
```
