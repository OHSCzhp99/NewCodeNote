## Redis
---
### key的层级
```java
项目名:业务名:类型:id （在图形可视化中会分为文件夹）

例如：
user相关的key：heima:user:1
product相关的key：heima:product:1
```

### 1. String 字符串
```java
1.set name zhp  设置键值对，key为name，value为zhp
2.get name      根据key获取的value值
3.mset name zhp age 25  一次设置多个键值对
4.mget name age         一次获取多个键值对
5.del name age          删掉一个或多个key

6.incr age              让key为age的值自增1
7.incrby age 2          让key为age的值自增2
8.incrbyfloat age 2.5   让key为age的值自增2.5

9.setnx name haip       设置键值对，但是已经有该key则不执行
10.setex name2 10 zhp    设置键值对并设置有效期，有效期要接在key后
11.expire name2 10      单独为key设置有效期
12.ttl name2            获取有效期剩余时间
```

### 2. Hash 哈希
```java
1.hset hash1 name zhp    设置key为hash1，里面字段为name，值为zhp
2.hget hash1 name        获取key为hash1，里面字段为name的值
3.hmset hash1 name zhp age 25  一次设置多个hash字段
4.hmget hash1 name age         一次获取多个hash字段 
5.hdel hash1 name age          删掉一个或多个key

6.hgetall hash1                一次获取所有hash的字段和值
7.hkeys hash1                  一次获取hash里的所有字段
8.havls hash1                  一次获取hash里的所有值

9.hincrby hash1 age 2          让hash中的字段自增2
10.hsetnx hash1 name zhp        设置hash的字段和值，但是已经有该字段则不执行
```

### 3. List 有序重复双向链表
```java
1.lpush user 1 2 3        向左侧分别插入1 2 3，结果为321
2.rpush user 4 5 6        向右侧分别插入4 5 6，结果为456
3.lpop user 2             删掉左侧两个元素
4.rpop user 2             删掉右侧两个元素

5.lrange user 0 -1        获取第一个到最后一个元素
6.llen user               查看列表长度
```

### 4. Set 无序不可重复
```java
1.sadd s1 a b c        插入key为s1，元素有a b c
2.srem s1 a            删除某个元素

3.smembers s1          输出全部元素
4.scard s1             返回元素个数
5.sismember s1 a       判断是否有某个元素

6.sinter s1 s2         返回key1和key2的交集
7.sdiff s1 s2          返回key1和key2的差集
8.sunion s1 s2         返回key1和key2的并集
```

### 5. SortedSet 有序不可重复
```java
1.zadd stus 85 jack 89 lucy   插入key为stus，学生的分数，分数作为score，学生作为value
2.zrem stues jack             删除元素

3.zrank stus jack             返回该下标，升序
4.zrank stus 0 2              返回下标范围内的元素
5.zrevrank stus jack          放回该下标，降序在z后加rev

6.zcard stus                  返回元素个数
7.zcount stus 0 80            返回0到80的元素个数
8.zrangebyscore stus 0 80     返回0到80的元素

9.zincrby stus 2 jack         给元素自增2
```
