## 下午 - 试题1
---
### 1. 数据流图基本图形元素
```c
外部实体：（E 矩形）
人、物、外部系统
--------------------
数据存储：（D 扁矩形/缺边）
表、文件
--------------------
加工：（P 圆角矩形）
至少要有一个输入和输出
只有输入没有输出：黑洞
只有输出没有输入：白洞
输入数据不足以产生数据：灰洞
--------------------
数据流：（箭头）
数据流的起点和终点必须要有一个是加工
```

### 2. 解题
```c
问题一：直接在文字和子图中寻找实体E
问题二：直接在文字和子图中寻找加工P、数据存储D（看数据流名称判断表名）

问题三：
① 父图子图平衡（对比父子之间的数据流名称是否存在，没有则标记）
② 加工要有输出输入（在子图中的加工的箭头要有出入，没有则标记）
③ 数据守恒（根据文子信息，寻找缺失的数据流）


问题四：采用结构化语言描述
IF
    THEN
ELSE
    IF
        THEN
    ELSE
    ENDIF
ENDIF
---------------
答题格式：
数据流名称：生产计划    起点：D3或生产计划表    终点：3或生产
（一般数据流名称可能是：缺失了信息表，要更新信息表，缺失了加工到加工）
```
