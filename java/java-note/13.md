### 1. 反射
```java
反射就是获取类里面所有的信息，
也可以结合配置文件动态创建对象
```

### 2. 反射创建方式
```java
1.全类名：包名+类名 
Class clazz = Class.forName("com.itheima.myreflect.Student"); //在类的复制路径

2.类名.class
Class clazz2 = Student.class;

3.对象.getClass()
Student s = new Student();
Class clazz3 = s.getclass();
```

### 3. 反射获取类内部
```java
//获取包名、类名
clazz.getPackage().getName()//包名
clazz.getSimpleName()//类名
clazz.getName()//完整类名
-----------------------
1.获取构造方法：
Constructor[] cons = clazz.getConstructors() 返回所有公共构造方法对象的数组
Constructor[] cons = clazz.getDeclaredConstructors() 返回所有构造方法对象的数组（包括私有）
Constructor cons = clazz.getConstructor()  返回单个公共构造对象
Constructor cons = clazz.getDeclaredConstructor(String.class, int.class) 返回单个构造对象（包括私有）

int modifiers = cons.getModifiers() 返回修饰符（1为公共，2为私有）
Parameter[] parameters = cons.getParameters() 返回构造方法的所有参数

cons.setAccessible(true) 对权限不校验（才可调用私有的构造方法）
Student stu = (Student) cons.newInstance("张三",23) 创建
-----------------------
2.获取成员变量：
Field[] fields = clazz.getFields() 返回所有公共的成员变量
Field[] fields = clazz.getDeclaredFields() 返回所有的成员变量（包括私有）
Field field = clazz.getField() 返回单个公共的成员变量
Field field = clazz.getDeclaredField("name") 返回单个成员变量（包括私有）

int modifiers = field.getModifiers() 返回修饰符（1为公共，2为私有）
String n = field.getName() 返回变量名
Class<?> type = field.getType() 返回变量类型
-----------------------
3.成员方法:
Method[] methods = clazz.getMethods() 返回所有的方法（包含父类中所有的公共方法，wait、sleep等） 
Method[] methods = clazz.getDeclaredMethods() 返回所有的方法（不包含父类）
Method m =  getMethod("eat",String.class) 返回单个方法，第一个参数指明式那个方法，第二是类型
getDeclaredMethod()

getModifiers() 返回修饰符
getName() 返回变量名
getParameters() 返回参数
getExceptionTypes() 返回抛出的异常

invoke() 调用方法，有三个参数：调用者、传参、返回值（后面两个没有可以不写）
比如：
Student s = new Student();
m.setAccessible(true); //当该方法为私有，需要开启不校验
m.invoke(s,"汉堡包"); //当方法没有返回值
Object result = m.invoke(s,"汉堡包") //当方法有返回值
String result = (Studen) m.invoke(s,"汉堡包") //当知道方法的返回值类型
```
