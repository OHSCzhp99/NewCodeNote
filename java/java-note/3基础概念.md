### 3. 计算机中的数据存储
```java
二进制 B
八进制 O
十进制 D
十六进制 H

进制转换：（任意转十，十转任意）
1.任意转十：
公式：系数 * 基数的权次幂 相加
系数是每一位上的数
基数是当前进制数
权是从右往左依次012345....
--------------------------------------
2.十转任意：
公式：除基取余法（直到商为0，再将余数倒拼）
```
