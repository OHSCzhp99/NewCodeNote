### 1. 判断 switch
```java
case 可以包含多个结果，例如1,2,3,4,用逗号隔开就好

jdk12新写法：（就不用写break了 ，只有一条输出语句的话，花括号也可以胜略）
case 1 -> { }
default -> { }
```

### 2. 循环 for、while
```java
无限循环：
for(;;) { }
while(true) { } 
```

### 3. 跳出控制语句
```java
continue：跳出本次循环，继续下次循环
break：结束全部循环
```
